📐 สูตรทั้งหมดในบทไฟฟ้าเคมี

รวบรวมสูตรสำคัญทุกสูตรที่จำเป็นต้องรู้ในบทเรียนไฟฟ้าเคมี ม.5

⚡ แรงดันไฟฟ้ามาตรฐานของเซลล์
E°cell = E°cathode − E°anode

ถ้า E°cell > 0 ปฏิกิริยาเกิดเองได้ (spontaneous) | ถ้า E°cell < 0 ต้องใช้ไฟฟ้าบังคับ

🔋 พลังงานอิสระกิบส์กับแรงดันไฟฟ้า
ΔG° = −nFE°cell

n = จำนวนโมลอิเล็กตรอนที่ถ่ายเท | F = 96,485 C/mol (ค่าคงที่ฟาราเดย์) | ΔG° < 0 เกิดเอง

⚖️ สมการเนิร์นสต์ (Nernst Equation)
E = E° − (RT/nF) ln Q
E = E° − (0.0592/n) log Q ที่ 25°C

ใช้คำนวณแรงดันที่ไม่ใช่สภาวะมาตรฐาน | Q = reaction quotient

⚗️ กฎฟาราเดย์ (Faraday's Law)
m = (M × I × t) / (n × F)

m = มวลสาร(g) | M = มวลโมลาร์ | I = กระแส(A) | t = เวลา(s) | n = จำนวน e⁻ | F = 96,485 C/mol

🔗 ΔG° กับ Keq
ΔG° = −RT ln Keq
log Keq = nE°cell / 0.0592

ที่ 25°C | R = 8.314 J/mol·K | ใช้ทำนายค่าคงที่สมดุลจาก E°

🌊 ปริมาณประจุไฟฟ้า
Q = I × t

Q = ปริมาณประจุ (คูลอมบ์, C) | I = กระแสไฟฟ้า (แอมแปร์, A) | t = เวลา (วินาที, s)

🧪 ครึ่งปฏิกิริยา (Half Reactions)
ออกซิเดชัน: A → Aⁿ⁺ + ne⁻
รีดักชัน: Bⁿ⁺ + ne⁻ → B

จำนวน e⁻ ที่ให้ = จำนวน e⁻ ที่รับ | ดุลโดยวิธีครึ่งปฏิกิริยา

📊 กฎเลขออกซิเดชัน
ธาตุอิสระ: ON = 0
H (ทั่วไป) = +1 | H (hydride) = −1
O (ทั่วไป) = −2 | O (peroxide) = −1

ผลรวม ON ในโมเลกุลกลาง = 0 | ในไอออน = ประจุของไอออน

บทเรียนที่ 1

เลขออกซิเดชัน (Oxidation Number / Oxidation State)

เลขออกซิเดชัน (ON) คือ ค่าประจุไฟฟ้าที่สมมติขึ้นของไอออนหรืออะตอมของธาตุ โดยคิดจากจำนวนอิเล็กตรอนที่ให้หรือรับหรือใช้ร่วมกัน ในสารประกอบไอออนิกเลขออกซิเดชันตรงกับค่าประจุจริง ส่วนในสารประกอบโคเวเลนต์เป็นเพียงประจุสมมติ

หลักเกณฑ์การกำหนดเลขออกซิเดชัน 6 ข้อ

  • ข้อ 1: ธาตุอิสระทุกชนิด (Na, Zn, Cu, He, H₂, N₂, O₂, Cl₂, P₄, S₈) มีเลขออกซิเดชัน = 0
  • ข้อ 2: H ในสารประกอบทั่วไป (HCl, H₂O, H₂SO₄) = +1 แต่ใน hydride ของโลหะ (NaH, CaH₂) = −1
  • ข้อ 3: O ในสารประกอบทั่วไป = −2 | ใน peroxide (H₂O₂, BaO₂) = −1 | ใน superoxide = −½ | ใน OF₂ = +2
  • ข้อ 4: ไอออนอะตอมเดียว ON = ค่าประจุจริง (Mg²⁺ = +2, F⁻ = −1)
  • ข้อ 5: ไอออนหลายอะตอม ผลรวม ON = ประจุของไอออน (SO₄²⁻ รวม = −2, NH₄⁺ รวม = +1)
  • ข้อ 6: ในสารประกอบกลาง ผลบวก ON ทั้งหมด = 0

ข้อเพิ่มเติม

  • ธาตุหมู่ IA, IIA, IIIA มี ON = +1, +2, +3 ตามลำดับ
  • อโลหะส่วนใหญ่มีหลายค่า เช่น Cl ใน HCl, HClO, HClO₂, HClO₃, HClO₄ = −1, +1, +3, +5, +7
  • ธาตุแทรนซิชันส่วนใหญ่มีมากกว่าหนึ่งค่า เช่น Fe ใน FeO, Fe₂O₃ = +2, +3

ตัวอย่างการคำนวณ

จงหา ON ของ Cr ใน [Cr(H₂O)₄Cl₂]ClO₄ H��O = 0, Cl⁻ = −1, ClO₄⁻ = −1 ให้ Cr = A A + (0×4) + (−1×2) + (−1) = 0 A − 2 − 1 = 0 → A = +3 ∴ Cr มีเลขออกซิเดชัน = +3
บทเรียนที่ 2

ปฏิกิริยารีดอกซ์ (Redox Reaction)

ไฟฟ้าเคมีเป็นการศึกษาปฏิกิริยาเคมีที่ทำให้เกิดกระแสไฟฟ้า หากใช้การถ่ายเทอิเล็กตรอนเป็นเกณฑ์ แบ่งได้ 2 ประเภท:

  • ปฏิกิริยารีดอกซ์ (Redox) — มีการถ่ายเท e⁻
  • ปฏิกิริยานอนรีดอกซ์ (Nonredox) — ไม่มีการถ่ายเท e⁻

ออกซิเดชัน vs รีดักชัน

ออกซิเดชัน (Oxidation): สารให้ e⁻ → ON เพิ่มขึ้น Zn(s) → Zn²⁺(aq) + 2e⁻ [Zn = ตัวรีดิวซ์, Reducing agent] รีดักชัน (Reduction): สารรับ e⁻ → ON ลดลง Cu²⁺(aq) + 2e⁻ → Cu(s) [Cu²⁺ = ตัวออกซิไดส์, Oxidizing agent]
ปฏิกิริยาออกซิเดชันและรีดักชันต้องเกิดขึ้นพร้อมกันเสมอ ไม่สามารถเกิดแยกกันได้

การพิจารณาว่าเป็นปฏิกิริยารีดอกซ์หรือไม่

  • ถ้าเลขออกซิเดชันมีการเปลี่ยนแปลง (สารหนึ่ง ON ↑ และอีกสาร ON ↓) → เป็นรีดอกซ์
  • ถ้ามีธาตุอิสระอยู่ด้วย → เป็นรีดอกซ์เสมอ

ตัวอย่าง: Cu จุ่มใน AgNO₃

ออกซิเดชัน: Cu(s) → Cu²⁺(aq) + 2e⁻ รีดักชัน: 2Ag⁺(aq) + 2e⁻ → 2Ag(s) ───────────────────────────── ปฏิกิริยารวม: Cu(s) + 2Ag⁺(aq) → Cu²⁺(aq) + 2Ag(s)

ข้อสังเกต

  • อโลหะอิสระ — มักเป็นตัวออกซิไดส์ (ชอบรับ e⁻)
  • โลหะอิสระ — มักเป็นตัวรีดิวซ์ (เสีย e⁻ ได้ง่าย)
  • ไอออนบวก — มักเป็นตัวออกซิไดส์
  • ไอออนลบ — มักเป็นตัวรีดิวซ์
บทเรียนที่ 3

เซลล์กัลวานิก ตอนที่ 1 (Galvanic Cell)

เซลล์กัลวานิก (หรือเซลล์โวลตาอิก) คือเซลล์ไฟฟ้าเคมีที่เปลี่ยนพลังงานเคมีเป็นพลังงานไฟฟ้า โดยอาศัยปฏิกิริยารีดอกซ์ที่เกิดขึ้นเอง (spontaneous) เมื่อแยกตัวรีดิวซ์ออกจากตัวออกซิไดส์

องค์ประกอบของเซลล์กัลวานิก

  • ขั้วแอโนด (Anode) — เกิดออกซิเดชัน, ขั้วลบ (−), โลหะค่อยๆ ละลาย
  • ขั้วแคโทด (Cathode) — เกิดรีดักชัน, ขั้วบวก (+), โลหะตกตะกอนเพิ่มขึ้น
  • สะพานเกลือ (Salt Bridge) — รักษาความเป็นกลางทางไฟฟ้า บรรจุ KCl หรือ KNO₃
  • วงจรภายนอก — อิเล็กตรอนไหลจากแอโนด → แคโทด

Daniel Cell (Zn-Cu)

แอโนด (Zn): Zn(s) → Zn²⁺(aq) + 2e⁻ [ออกซิเดชัน] แคโทด (Cu): Cu²⁺(aq) + 2e⁻ → Cu(s) [รีดักชัน] ───────────────────────────────────── ปฏิกิริยารวม: Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s)

สะพานเกลือ

ไอออนบวก (K⁺) เคลื่อนที่สู่ครึ่งเซลล์รีดักชัน | ไอออนลบ (Cl⁻, NO₃⁻) เคลื่อนที่สู่ครึ่งเซลล์ออกซิเดชัน ทำให้ปฏิกิริยาดำเนินต่อไปได้นานขึ้น

ประเภทของเซลล์กัลวานิก

  • เซลล์ปฐมภูมิ (Primary cell) — ใช้แล้วทิ้ง
  • เซลล์ทุติยภูมิ (Secondary cell) — ชาร์จใหม่ได้
บทเรียนที่ 4

เซลล์กัลวานิก ตอนที่ 2 — แรงเคลื่อนไฟฟ้าและเซลล์แห้ง

แรงเคลื่อนไฟฟ้า (EMF)

ความต่างศักย์ระหว่างขั้วแอโนดและแคโทดเรียกว่า แรงเคลื่อนไฟฟ้า (electromotive force: emf) มีหน่วยเป็นโวลต์ (V)

E°cell = E°cathode − E°anode ถ้า E°cell > 0 → ปฏิกิริยาเกิดเองได้ (spontaneous) ถ้า E°cell < 0 → ต้องใช้ไฟฟ้าบังคับ

เซลล์แห้ง (Dry Cell / Leclanché Cell)

กล่องสังกะสี (Zn) ทำหน้าที่เป็นแอโนด ไส้ถ่าน (C) เป็นแคโทด อิเล็กโทรไลต์เป็นของแข็งกึ่งเหลว (NH₄Cl และ ZnCl₂ ผสมกับแป้ง)

แอโนด: Zn(s) → Zn²⁺ + 2e⁻ แคโทด: 2MnO₂ + 2NH₄⁺ + 2e⁻ → Mn₂O₃ + 2NH₃ + H₂O แรงดัน: ~1.5 V

เซลล์อัลคาไลน์ (Alkaline Battery)

แอโนด: Zn + 2OH⁻ → ZnO + H₂O + 2e⁻ แคโทด: MnO₂ + H₂O + 2e⁻ → Mn(OH)₂ + 2OH⁻ → ~1.5 V

เซลล์ทุติยภูมิ: แบตเตอรี่ตะกั่ว-กรด (Lead-Acid)

แอโนด: Pb + SO₄²⁻ → PbSO₄ + 2e⁻ แคโทด: PbO₂ + SO₄²⁻ + 4H⁺ + 2e⁻ → PbSO₄ + 2H₂O → ~2 V ต่อเซลล์
บทเรียนที่ 5

การเขียนแผนภาพเซลล์กัลวานิก (Cell Notation)

การเขียนแผนภาพเซลล์เป็นรูปแบบย่อสำหรับแสดงองค์ประกอบของเซลล์กัลวานิก

หลักการเขียน

  • เขียนสารที่ขั้วแอโนดทางซ้าย (ออกซิเดชัน)
  • เขียนสารที่ขั้วแคโทดทางขวา (รีดักชัน)
  • ใช้ | (เส้นเดี่ยว) แทนขอบเขตระหว่างเฟส
  • ใช้ || (เส้นคู่) แทนสะพานเกลือ
  • ถ้าใช้ขั้วเฉื่อย เช่น Pt ให้ระบุก่อนสารในครึ่งเซลล์
รูปแบบ: แอโนด | สารละลายแอโนด || สารละลายแคโทด | แคโทด ตัวอย่าง 1 (Daniel Cell): Zn(s) | Zn²⁺(aq, 1M) || Cu²⁺(aq, 1M) | Cu(s) E° = +1.10 V ตัวอย่าง 2 (ขั้วเฉื่อย Pt): Pt | H₂(g, 1atm) | H⁺(aq, 1M) || Cu²⁺(aq, 1M) | Cu(s) ตัวอย่าง 3 (หลายไอออนในครึ่งเซลล์): Pt | Fe²⁺(aq), Fe³⁺(aq) || Ag⁺(aq) | Ag(s)
บทเรียนที่ 6

เซลล์อิเล็กโทรไลต์ (Electrolytic Cell)

เซลล์อิเล็กโทรไลต์ คือเซลล์ที่ต้องใช้พลังงานไฟฟ้าจากภายนอกบังคับให้ปฏิกิริยาที่ไม่เกิดขึ้นเองเกิดขึ้น (nonspontaneous: E°cell < 0, ΔG° > 0)

เปรียบเทียบเซลล์กัลวานิก vs เซลล์อิเล็กโทรไลต์

  • เซลล์กัลวานิก: เคมี → ไฟฟ้า | E°cell > 0 | ΔG° < 0
  • เซลล์อิเล็กโทรไลต์: ไฟฟ้า → เคมี | E°cell < 0 | ΔG° > 0

การชุบโลหะด้วยไฟฟ้า (Electroplating)

  • แคโทด = วัตถุที่ต้องการชุบ (โลหะตกตะกอนลงบน)
  • แอโนด = โลหะที่ใช้ชุบ (ละลายเข้าไปแทน)
  • จุ่มในสารละลายเกลือของโลหะที่ต้องการชุบ
มวลโลหะที่ชุบได้ (กฎฟาราเดย์): m = (M × I × t) / (n × F) M = มวลโมลาร์ (g/mol) n = จำนวนอิเล็กตรอนที่ถ่ายเทต่อไอออน F = 96,485 C/mol I = กระแส (A), t = เวลา (s)
บทเรียนที่ 7

การแยกสารละลายด้วยไฟฟ้า (Electrolysis)

อิเล็กโทรลิซิส คือกระบวนการใช้กระแสไฟฟ้าทำให้สารเกิดการสลายตัว

การแยกน้ำ (Water Electrolysis)

แคโทด (รีดักชัน): 2H₂O + 2e⁻ → H₂(g) + 2OH⁻(aq) แอโนด (ออกซิเดชัน): 2H₂O → O₂(g) + 4H⁺(aq) + 4e⁻ ────────────────────────────────────────── สมการรวม: 2H₂O → 2H₂(g) + O₂(g)

การแยกสารละลาย NaCl

แคโทด: 2H₂O + 2e⁻ → H₂(g) + 2OH⁻ → ได้ H₂ แอโนด: 2Cl⁻ → Cl₂(g) + 2e⁻ → ได้ Cl₂ สารละลาย: Na⁺ + OH⁻ → NaOH ผลิตภัณฑ์: H₂, Cl₂, NaOH

การแยก NaCl หลอมเหลว

แคโทด: Na⁺ + e⁻ → Na(l) แอโนด: 2Cl⁻ → Cl₂(g) + 2e⁻

กฎฟาราเดย์

m = (M × I × t) / (n × F) ตัวอย่าง: ชุบ Cu ด้วยกระแส 2A เป็นเวลา 30 นาที (M=63.5, n=2) m = (63.5 × 2 × 1800) / (2 × 96485) = 1.18 g
บทเรียนที่ 8

เซลล์กัลวานิก (ทบทวน)

สรุปหลักสำคัญเซลล์กัลวานิก

  • แอโนด (−): ออกซิเดชัน, โลหะละลาย (มวลลด), ไอออนบวกเพิ่ม
  • แคโทด (+): รีดักชัน, โลหะตกตะกอน (มวลเพิ่ม), ไอออนบวกลด
  • e⁻ ไหล: แอโนด → แคโทด (ผ่านสายไฟ)
  • ไอออนไหล: ผ่านสะพานเกลือ (K⁺ → แคโทด, Cl⁻ → แอโนด)

ความสัมพันธ์สำคัญ

E°cell = E°cathode − E°anode E°cell > 0 → ΔG° < 0 → K > 1 (เกิดเองได้) E°cell < 0 → ΔG° > 0 → K < 1 (ไม่เกิดเอง) E°cell = 0 → ΔG° = 0 → K = 1 (สมดุล) ΔG° = −nFE°cell log K = nE°cell / 0.0592 (ที่ 25°C)

การเลือกแอโนด-แคโทด

โลหะที่มี E° ต่ำกว่า (ค่าลบมากกว่า) จะเป็น แอโนด (ออกซิไดส์ง่ายกว่า) โลหะที่มี E° สูงกว่าจะเป็น แคโทด

บทเรียนที่ 9

ศักย์ไฟฟ้าครึ่งเซลล์มาตรฐาน (Standard Electrode Potential, E°)

ค่า E° วัดโดยเทียบกับขั้วไฮโดรเจนมาตรฐาน (SHE) ซึ่งกำหนดให้ E° = 0.000 V ที่ 25°C, ความดัน 1 atm, ความเข้มข้น 1 M

ขั้วไฮโดรเจนมาตรฐาน (SHE): 2H⁺(aq, 1M) + 2e⁻ ⇌ H₂(g, 1atm) E° = 0.000 V

ตารางค่า E° (ปฏิกิริยารีดักชัน) ที่สำคัญ

F₂(g) + 2e⁻ → 2F⁻(aq) E° = +2.87 V (ตัวออกซิไดส์แรงที่สุด) Au³⁺ + 3e⁻ → Au(s) E° = +1.50 V Cl₂(g) + 2e⁻ → 2Cl⁻(aq) E° = +1.36 V O₂ + 4H⁺ + 4e⁻ → 2H₂O E° = +1.23 V Ag⁺ + e⁻ → Ag(s) E° = +0.80 V Fe³⁺ + e⁻ → Fe²⁺ E° = +0.77 V Cu²⁺ + 2e⁻ → Cu(s) E° = +0.34 V 2H⁺ + 2e⁻ → H₂(g) E° = 0.00 V (SHE มาตรฐาน) Fe²⁺ + 2e⁻ → Fe(s) E° = −0.44 V Zn²⁺ + 2e⁻ → Zn(s) E° = −0.76 V Al³⁺ + 3e⁻ → Al(s) E° = −1.66 V Mg²⁺ + 2e⁻ → Mg(s) E° = −2.37 V Na⁺ + e⁻ → Na(s) E° = −2.71 V Li⁺ + e⁻ → Li(s) E° = −3.05 V (ตัวรีดิวซ์แรงที่สุด)

ค่า E° สูง → รีดิวซ์ได้ง่าย (ตัวออกซิไดส์แรง) | ค่า E° ต่ำ → ออกซิไดส์ได้ง่าย (ตัวรีดิวซ์แรง)

บทเรียนที่ 10

ประโยชน์ของค่า E° (Applications of Standard Electrode Potential)

1. ทำนายทิศทางปฏิกิริยา

E°cell = E°(ขั้วที่ E° สูงกว่า) − E°(ขั้วที่ E° ต่ำกว่า) ถ้า E°cell > 0 → ปฏิกิริยาเกิดเอง (spontaneous) ถ้า E°cell < 0 → ปฏิกิริยาไม่เกิดเอง

2. เปรียบเทียบความแรงตัวออกซิไดส์/ตัวรีดิวซ์

  • E° สูงกว่า → เป็นตัวออกซิไดส์ที่แรงกว่า
  • E° ต่ำกว่า → เป็นตัวรีดิวซ์ที่แรงกว่า

3. หาค่าคงที่สมดุล Keq

log Keq = nE°cell / 0.0592 (ที่ 25°C) หรือ ΔG° = −nFE° = −RT ln Keq

4. สมการเนิร์นสต์ (ไม่ใช่สภาวะมาตรฐาน)

E = E° − (0.0592 / n) × log Q (ที่ 25°C) โดย Q = reaction quotient = [ผลิตภัณฑ์] / [สารตั้งต้น]

ตัวอย่าง

เซลล์ Zn-Cu: E°cell = E°(Cu²⁺/Cu) − E°(Zn²⁺/Zn) = 0.34 − (−0.76) = +1.10 V log K = (2 × 1.10) / 0.0592 = 37.2 K = 10^37.2 → ปฏิกิริยาเกิดเองได้มาก
บทเรียนที่ 11

เซลล์ปฐมภูมิ (Primary Cell)

เซลล์ปฐมภูมิคือเซลล์กัลวานิกที่ใช้แล้วทิ้ง ชาร์จใหม่ไม่ได้

เซลล์แห้ง (Leclanché Cell / Dry Cell)

แอโนด: Zn(s) → Zn²⁺ + 2e⁻ แคโทด: 2MnO₂ + 2NH₄⁺ + 2e⁻ → Mn₂O₃ + 2NH₃ + H₂O แรงดัน: ~1.5 V | อิเล็กโทรไลต์: NH₄Cl + ZnCl₂

ถ่านอัลคาไลน์ (Alkaline Battery)

แอโนด: Zn(s) + 2OH⁻ → ZnO + H₂O + 2e⁻ แคโทด: 2MnO₂ + H₂O + 2e⁻ → Mn₂O₃ + 2OH⁻ แรงดัน: ~1.5 V | อิเล็กโทรไลต์: KOH (ด่าง) | จ่ายกระแสได้นานกว่าถ่านธรรมดา

เซลล์ปรอท (Mercury Cell)

แอโนด: Zn + 2OH⁻ → ZnO + H₂O + 2e⁻ แคโทด: HgO + H₂O + 2e⁻ → Hg(l) + 2OH⁻ แรงดัน: ~1.35 V (คงที่มาก) | ใช้ในนาฬิกา เครื่องช่วยฟัง

เซลล์เชื้อเพลิง H₂-O₂ (Fuel Cell)

แอโนด: H₂(g) + 2OH⁻ → 2H₂O + 2e⁻ แคโทด: O₂(g) + 2H₂O + 4e⁻ → 4OH⁻ สมการรวม: 2H₂ + O₂ → 2H₂O แรงดัน: ~1.23 V | ประสิทธิภาพสูง ไม่มีมลพิษ ใช้ในยานอวกาศ
บทเรียนที่ 12

การผุกร่อนของโลหะและการป้องกัน (Corrosion & Prevention)

การผุกร่อน (Corrosion) คือกระบวนการที่โลหะถูกออกซิไดส์โดยสิ่งแวดล้อม ทำให้เกิดสนิมหรือโลหะออกไซด์

กลไกการเกิดสนิมเหล็ก

เหล็กทำหน้าที่เป็นทั้งแอโนดและแคโทดในบริเวณต่างกัน (electrochemical corrosion)

แอโนด (บริเวณรอยแตก): Fe(s) → Fe²⁺(aq) + 2e⁻ แคโทด (บริเวณน้ำ/O₂): O₂(g) + 2H₂O + 4e⁻ → 4OH⁻(aq) ────────────────────────────────────────── Fe²⁺ + 2OH⁻ → Fe(OH)₂ → Fe₂O₃·nH₂O (สนิมแดง)

ปัจจัยที่เร่งการผุกร่อน

  • ความชื้น/น้ำ — เป็นตัวนำไฟฟ้า
  • O₂ — ตัวออกซิไดส์
  • กรด (H⁺) — เร่งปฏิกิริยา
  • เกลือ (Cl⁻, SO₄²⁻) — เพิ่มการนำไฟฟ้า
  • โลหะต่างชนิดสัมผัสกัน — เกิด galvanic corrosion

วิธีป้องกันการผุกร่อน

  • เคลือบผิว — สี, พลาสติก, ยาง
  • Galvanization — ชุบสังกะสี (Zn ออกซิไดส์แทน Fe เพราะ E°Zn < E°Fe)
  • Tinning — ชุบดีบุก (ป้องกันทางกายภาพ)
  • Cathodic Protection — ต่อ Zn หรือ Mg กับโลหะที่ต้องการ (sacrificial anode)
  • Impressed Current — ใช้กระแสไฟฟ้าภายนอกทำให้เป็นแคโทด
  • โลหะผสม — สแตนเลส (Fe + Cr + Ni) ทนสนิม
